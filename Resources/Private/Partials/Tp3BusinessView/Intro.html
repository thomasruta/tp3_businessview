Sollten Sie diesen Bildschirm sehen - fehlt die StoragePid
{$plugin.tx_tp3businessview_tp3businessview.persistence.storagePid} tragen sie diese nachdem das statische Template eingefügt wurde im roottemplate bei den konstanten ein.

Nach der Installation sollte man API Keys bei google anlegen, diese werden für Places Suche im Backend benötigt.

https://console.developers.google.com/apis/

Folgende Apis werden empfohlen:

  • Google Maps JavaScript API
  • Google Maps Geocoding API
  • Google Street View Image API
  • Street View Publish API (für kommende Features)
